# フライトシミュレーターデモ


## テクノロジースタック
- Nextjs
- TypeScript
- Styled Component 
- Material UI
- [recharts](https://recharts.org/en-US/)
- Node(v10) 
  - 最新LTSのv14系でも動くはずです

## インフラ
- Firebase Hostringを使用しています
- CI/CDはありません
  - おすすめは[こちら](https://firebase.google.com/docs/hosting/github-integration)

## ローカルセットアップ

```bash
$ npm install

$ npm run dev

$ open localhost:3000
``` 


## デプロイ
```bash
$ npm run deploy
```
